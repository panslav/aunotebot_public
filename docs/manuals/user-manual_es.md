# Manual de usuario de AUNotebot

## 1. Inicio
1. Abre el chat del bot.
2. Envía `/start`.
3. En la pantalla inicial verás: `Buscar`, `Tareas`, `Notas`, `Recordatorios`, `Borradores`, `Más`.

## 2. Idioma
- Menú: `More -> Settings -> Idioma`.
- Idiomas: English, Русский, Українська, Español, Português, Français.

## 3. Crear una nota
1. Envía texto/voz/video/imagen.
2. El bot crea un borrador.
3. Pulsa `📝 Guardar como nota`.

## 4. Crear una tarea
- En borrador: `🟧 Guardar como tarea`.
- O en una nota: `⬛ Make Task`.

## 5. Reenvío de mensajes
- Varios mensajes reenviados en una sola acción se unen en un único borrador.
- El tiempo de unión se configura en `More -> Settings -> Tiempo de unión de mensajes múltiples`.
- Hashtags enviados aparte se agregan como etiquetas al borrador.

## 6. Búsqueda y Recuerdo
- `🔎 Find` activa modo interactivo.
- Envía una consulta en texto libre.
- El bot devuelve un resumen y botones con los elementos más relevantes.

## 7. Acciones en elemento
### Nota
- `✏️ Edit`, `🏷️ Tags`, `⬛ Make Task`, `❌ Remove`, `⬅️ Back`

### Tarea
- `✏️ Edit`, `✅ Done/🔄 Re-do`, `⏰ Reminder`, `🏷️ Tags`, `🔁 Make Note`, `🔀 Merge`, `👥 Assign`, `❌ Remove`, `⬅️ Back`

## 8. Etiquetas y temas
- `Tags` abre las etiquetas del elemento.
- Al pulsar una etiqueta se abre la lista de elementos relacionados.
- En el menú del tema: `Edit` y `Delete`.

## 9. Recordatorios
- En una tarea pulsa `⏰ Reminder`.
- Escribe fecha/hora en formato libre (ejemplo: «mañana 9:00»).

## 10. Ajustes
`More -> Settings`:
- Idioma
- Zona horaria
- Al marcar como hecho
- Copiar texto para editar
- Resumen diario de tareas
- Resumen de la tarde
- Tiempo de unión de mensajes múltiples
- N.º de elementos
- Mostrar menú inferior

## 11. Ayuda y legal
- `More -> Help` abre este manual en la web.
- `More -> Legal` abre los Terms & Conditions en tu idioma actual.

## 12. Contacto
- Ideas y feedback: `@aunotebot_support`
