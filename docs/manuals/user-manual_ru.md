# Руководство пользователя AUNotebot

## 1. Старт
1. Откройте чат с ботом.
2. Отправьте `/start`.
3. На стартовом экране доступны: `Поиск`, `Задачи`, `Заметки`, `Напоминания`, `Черновики`, `Ещё`.

## 2. Язык
- Меню: `More -> Settings -> Язык`.
- Поддерживаются: English, Русский, Українська, Español, Português, Français.

## 2a. AI-баланс и Telegram Stars
- AI-функции используют ваш баланс баллов.
- Модель по умолчанию:
  - `1 балл = 1 Telegram Star`
  - `50 бесплатных баллов` при первом запуске
  - `5 бесплатных баллов` каждый день в `00:00` по вашему локальному часовому поясу
  - `0,1 балла = 1 AI-действие`
- Откройте `More -> Points`, чтобы посмотреть баланс, текущую стоимость AI-действия, последние 5 операций с баллами, открыть `Полную историю баллов` и пополнить баланс через Telegram Stars.
- Если баланса недостаточно, конкретное AI-действие блокируется, но не-AI функции продолжают работать.

## 3. Создание заметки
1. Отправьте текст/голос/видео/изображение.
2. Бот создаст черновик.
3. Нажмите `📝 Сохранить как заметку`.

### Ограничения для голоса, аудио и видео
- Голосовые сообщения, аудиофайлы, видео, video note и видео-документы из Telegram обрабатываются только если размер файла не превышает `20 МБ`.
- Это ограничение задаётся не OpenAI, а транспортом Telegram Bot API: бот не может скачать более крупный файл для расшифровки.
- Если файл больше `20 МБ`, бот сразу покажет ошибку и не создаст черновик.
- Для более длинной записи или видео нужно заранее разбить файл на части или сжать его и отправить снова.

## 4. Создание задачи
- В черновике: `🟧 Сохранить как задачу`.
- Или в карточке заметки: `⬛ Make Task`.

## 5. Пересылка сообщений
- Несколько пересланных сообщений в одном действии объединяются в один черновик.
- Окно объединения настраивается: `More -> Settings -> Время объединения нескольких сообщений`.
- Хэштеги в отдельном сообщении добавляются в теги черновика.

## 6. Поиск и Сводка
- Кнопка `🔎 Find` включает интерактивный режим.
- Отправьте вопрос в свободной форме.
- Бот вернет краткий ответ и список релевантных элементов кнопками.

## 7. Действия в карточке
### Заметка
- `✏️ Edit`, `🏷️ Tags`, `⬛ Make Task`, `❌ Remove`, `⬅️ Back`

### Задача
- `✏️ Edit`, `✅ Done/🔄 Re-do`, `⏰ Reminder`, `🏷️ Tags`, `🔁 Make Note`, `🔀 Merge`, `👥 Assign`, `❌ Remove`, `⬅️ Back`

## 8. Теги и темы
- `Tags` открывает теги элемента.
- По нажатию на тег открывается список связанных элементов.
- В меню темы доступны `Edit` и `Delete`.

## 9. Напоминания
- В задаче нажмите `⏰ Reminder`.
- Введите дату/время в свободной форме (например, «завтра 9:00»).

## 10. Настройки
`More -> Settings`:
- Язык
- Часовой пояс
- При отметке как выполнено
- Копировать текст для редактирования
- Ежедневная сводка задач
- Вечерняя сводка
- Время объединения нескольких сообщений
- Кол-во элементов
- Показывать нижнее меню

## 11. Помощь и правовые документы
- `More -> Help` открывает это руководство в веб-версии.
- `More -> Legal` открывает Terms & Conditions на текущем языке интерфейса.
